2. Статика

Статичные страницы и слайдер.

2.1. Статичные страницы

Статичные страницы представляют из себя набор готовых страниц с заданным URL адресом. Список доступных страниц: О нас, Оплата, Политика конфиденциальности, Доставка, Контакты, Как заказать, Скидки и бонусы, Возврат товаров.

В настройках каждой страницы есть возможность отобразить ссылку на нее как в футере, так и в хедере сайта, а так же сделать ее полностью скрытой. Дополнительные поля позволяют настроить метатеги и изображение страницы.

2.2. Слайдер

Слайдер на главной странице может содержать неограниченное количесво слайдов. Слайды можно сделать с описаниями и ссылками на посадочные страницы, для этого достаточно заполнить необходимые поля.

Настройки слайдера позаоляют включить и выключить такие пункаты как: стрелки, точки навигации, автолистание. Также можно настроить скорость анимации пролистывания в милисекундах.

3. Магазин

Работа с товарами и разделом магазина сайта.

3.1 Настройки

3.1.1 Настройки магазина

Описание пунктов настройки:

Время работы - отображается в шапке сайта рядом с телефоном.

Время начала доставки - с этого времени (+ задержка при формировании заказа) будет считаться доставка в предлагаемом при оформлении заказа списке.

Время окончания доставки - до этого времени рассчитывается доставка.

Доставка 24/7 - при активном пункте меню убирает ограничения по времени.

Задержка при формировании заказа на доставку - сколько времени необходимо на приготовление и доставку заказа.

Доставка только при оплате онлайн - запрещает доставку при оплате другими способами.

Включить зоны доставки - включает рассчет стоимости доставки по активным зонам доставки (необходима загрузка файла с зонами доставки)

Стоимость доставки - единая стоимость доставки (не работает при включенных зонах доставки)

Сумма заказа для бесплатной доставки - обнуляет стоимость доставки после заданной суммы.

Показать короткое описание товара под карточкой - выводит короткое описание товара в списке товаров.



Товар при 0 остатке - не актуально для магазинов в тематике "Еда"

Описание каталога - описание общего каталога (меню). Расположено внизу страницы /catalog/

H1 - главный заголовок каталога.

Мета тайтл, Мета описание, Ключевые слова - настройки метатегов для страницы каталога.

Изображение - использыется при репосте страницы в соцсетях (сниппет с изображением)

Включить акции, Включить новинки, Включить хиты продаж - активирует или скрывает соответствующие блоки на главной странице сайта.

3.3 Точки самовывоза

Если у вас несколько пунктов самовывоза, укажите их. Список пунктов появится при оформлении заказа. Также контакты пунктов самовывоза можно вывести в подвале сайта.

3.4 Способы оплаты

Заменяют способы оплаты "по-умолчанию".



3.2 Категории

Позволяет создавать, редактировать, удалять категории товаров. Стоит обратить внимание на пункт "Отображать в меню". Если этот пункт не отмечен, то категория не отображается в шапке сайта.

3.3 Товары

Позволяет создавать, редактировать, удалять товары. Обязательные пункты для заполнения обозначены *.

3.4 Сопутствующие товары

Сопутствующие товары можно привязать к конкретной категории или ко всем категориям. В сопутствующие товары можно включить такие товары, как столовые приборы. Сопутствующий товар будет автоматически добавлться в корзину при добавлении любого товара или товара, относящегося к нужной категории. Можно настроить количество бесплатных позиций и минимальное количество товаров для заказа.

3.5 Характеристики

При создании характеристик, они становятся доступны для заполнения в разделе создания/рудактирования товара. Характеристики отображаются списком в карточке товара.

4. Продажи

4.1 Покупатели

Список всех покупателей. Возможность просмотреть все заказы, которые совершик конкретный покупатель.

4.2 Заказы

Список всех заказов с описаниями. При открытии карточки заказа есть возможность изменить статус заказа (если пользователь зарегистрирован, статус будет отображаться в ЛК).

6. Маркетинг

Промокоды и карты лояльности.

6.1 Промокоды

Устанавливайте скидки по промокодам. Скидка настраивается в %

6.2 Карты лояльности

Карты лояльности начинают работать только после включения "Оповещения по смс" для предотвращения использования скидок и бонусов без подтверждения номера телефона. Перейдите в "Настройки->Общие настройки" и установите флажок на пункте "Разрешить отправку смс сообщений для регистрации и подтверждения заказов", сохраните изменение настроек. Также необходимо задать уровни лояльности в настройках. Если скидка одна для всех, без повышения, то создайте один уровень.
